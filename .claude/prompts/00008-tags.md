## Feature 1: Admin support

We need a way to identify some users as admins. Admins can mark other users as admins. I don't need roles or 
permissions. Admins will just be able to do things that I code for admins. Each time the app is bootstrapped, there 
should be a way to identify one user as an initial admin. 

### Admin features

In this commit, the only admin feature: when viewing a profile (performer a coach), an admin can make that user an 
admin.

### Verification

- Make sure the admin user can turn other users into admins
- Make sure another admin user can also create admins

## Feature 2: Tag support

Let's add tags to:
- performers
- coaches
- teams

For now, let's assume all tags are specific to their "domain" (e.g. performer tags are separate from coach tags). 

Overall, the way tags work is that they are arbitrary strings user can add to these entities. When editing profiles, 
users can add or remove tags using an auto-complete (if it's easy, maybe we can suggest tags at the top)? They can 
also type a new tag to suggest it. New tags must be approved by an admin.

### Tag approval

Let's add a section to the approvals page for admins that allows tag approval. Only 1 admin needs to approve. This 
should cut down on spam or offensive tags. Let's also list the page the user who suggested the tag wants to use this 
tag on as context can be helpful.

### Tags on profile pages

I'll let you take a pass on where best in the design to list tags

### Search

All search pages (performers, coaches, teams) should allow filtering by tags. This is where the real power of tags 
comes in. Users can also use the same auto-complete interface to add one or more tags. Like all search filters, 
these should be "AND" filters.

### Verification

- Use the preview browser
- Add some tags to a performer profile, including new tags
- View the profile, they should not appear yet
- Switch to an admin and approve the tag(s)
- Now switch back and view the profile, you should see the tag
- Try searching and the tag should auto-complete and find the appropriate profile
- Do the same for coaches and teams
